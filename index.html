📲 Mobile Portfolio App Setup Guide

Aap apne mobile phone se direct stocks save aur delete karne ke liye ek personal **Google Web App** deploy kar sakte hain jo bina kisi server ke directly aapke phone par chalega aur Google Sheet data updates karega:

  1. Apne Google Sheet ko open karein.
  2. Top menu me Extensions -> Apps Script par click karein.
  3. Wahan editor me pehle se likha code delete karke google-apps-script.js file ka code paste karein.
  4. Apps Script editor me left sidebar par + (Add a file) par click karke HTML file select karein aur uska naam Index rakhein. Usme google-apps-script-index.html ka code paste karein.
  5. Top right corner me Deploy -> New deployment par click karein.
  6. Select type me gear icon par click karke Web app choose karein.
  7. Execute as: "Me" aur Who has access: "Anyone" select karke deploy karein.
  8. Mili hui Web app URL ko copy karke yahan input section me paste karein aur save karein! Ab aap is website ko bina kisi PC server ke GitHub Pages par host kar sakte hain.
